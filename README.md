# Traffic Light Controller using Verilog HDL

## Objective
Design and simulate a traffic light controller using Finite State Machine (FSM) concepts in Verilog HDL.

## Features
- Timer-based traffic signal control
- FSM implementation
- Verilog HDL design
- Simulation using EDA Playground

## Tools Used
- Verilog HDL
- EDA Playground
- GitHub
